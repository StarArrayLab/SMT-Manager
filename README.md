# SMT-Manager
SMT物料管理程序
# 中文说明
  做这个程序是因为我自己个人有了贴片机，加上买物料整盘购买需要有个管理程序好让我知道自己有多少物料，以及都是什么东西，什么参数和规格，避免以后购买重复花冤枉钱，目前只做了部分器件的支持，如果没有你的器件类型可以联系我xie_sat@163.com，在之后的版本我会改进并添加。

  V2.0将会支持技术手册定向功能，也就是在本地定向你已有的器件技术手册，等后面我有时间再做，毕竟目前个人觉得用不到，大家画板子都会在项目文件里放入“技术手册”文件夹，而且立创商城也很容易找手册。

  需要在cmd中安装pip install PyQt5
# English instructions
  I created this program because I have my own pick-and-place machine, and I need a management system for purchasing materials in bulk to keep track of how many materials I have, what they are, their parameters and specifications, and to avoid unnecessary expenses from duplicate purchases. Currently, only some components are supported. If your component type is not listed, please contact me at xie_sat@163.com, and I will improve and add it in a later version.

  Version 2.0 will support a technical manual targeting function, allowing you to target existing component technical manuals locally. I'll implement this later when I have time, as I personally don't think it's necessary right now. Everyone who designs PCBs will put a "Technical Manual" folder in their project files, and manuals are easily found on the LCSC online store.

  需要在cmd中安装pip install PyQt5
