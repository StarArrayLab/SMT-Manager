SMT物料管理程序

中文说明
  
  做这个程序是因为我自己个人有了贴片机，加上买物料整盘购买需要有个管理程序好让我知道自己有多少物料，以及都是什么东西，什么参数和规格，避免以后购买重复花冤枉钱，目前只做了部分器件的支持，如果没有你的器件类型可以联系我xie_sat@163.com，在之后的版本我会改进并添加。

  Files里放的是V1.0的代码，现在V2.0已更新，增加技术手册定向，购买链接定向，已有物料搜索，更换技术栈，减小存储压力，可导入/导出目前已有的物料库，新增几种物料种类，解决内存溢出问题，可至少存储上百条数据。

  V3.0正在开发，将会是手机版发布，手机版需与电脑版用蓝牙对接数据，数据均保存本地。

English instructions
  
  This program was developed because I own a pick-and-place machine and need a management system for purchasing materials in bulk. This allows me to track the amount of materials I own, their types, parameters, and specifications, preventing unnecessary purchases and wasting money. Currently, only some components are supported. If your component type is not listed, please contact me at xie_sat@163.com. I will improve and add support in future versions.

  The Files folder contains the code for version 1.0. Version 2.0 has been updated, adding targeted access to technical manuals and purchase links, existing material search, a changed technology stack, reduced storage pressure, import/export of existing material libraries, addition of several new material types, and resolution of memory overflow issues. It can now store at least hundreds of data entries.

  Version 3.0 is under development and will be released as a mobile version. The mobile version will connect to the PC version via Bluetooth for data exchange, and all data will be stored locally.
